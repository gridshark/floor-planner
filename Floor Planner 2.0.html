How to Use

  1. Set room size and click Apply
  2. Name & size an object, pick a color, click Add
  3. Drag objects to reposition (snaps to ½ ft grid)
  4. Click an object to select it
  5. Use the ↻ button in the object list or press R to rotate 45°
  6. Click the border frame to mark windows (yellow)
  7. ⌘C / ⌘V to copy & paste selected object
  8. ⌘Z / ⌘⇧Z to undo & redo
  9. Use 💾 to save and 📁 to load a floor plan
  10. Arrow keys nudge the selected object by ½ ft
